How It Works

1

Aerial Imagery

Fetches satellite/aerial photos from Google Tiles API. Stitches multiple 256px tiles into a single high-resolution image. Auto-selects zoom level to stay within memory limits.

Google Tiles APIPillow
2

OSM Data

Retrieves building footprints, roads, driveways, address polygons, and property boundaries from OpenStreetMap via the Overpass API. All free, no API key needed.

Overpass APIGeoPandasShapely
3

Vegetation Detection

Multi-range HSV colour segmentation catches all grass types (healthy, dry, shadowed). Excess Green Index confirmation. Texture-based grass/tree separation using coefficient of variation.

OpenCVHSV + ExGNumPy
4

Classification

STRtree spatial indexes for O(log n) road lookups. Address-matching, driveway-matching, road-fallback. Consensus correction + outlier detection. Driveway override.

STRtreeSciPyHeuristics
5

Pin Placement

One front + one back pin per building. 4-attempt fallback cascade. Property border enforcement, lateral corridor constraint, tree canopy penalty, boundary deduplication.

Scoring 0-100Grass/Paved/Drive
6

Output

Segmentation mask, delivery pins with scores, map visualisation, REST API responses. Precomputed results cached for instant lookups.

FastAPICachePNG Maps

API Reference

GET /health

Health check. Returns status, timestamp, and API version.

POST /api/garden-pins

Get front and back garden delivery pins for a single house. Accepts lat/lon, a coords string, or an Irish Eircode (geocoded via 5-strategy cascade).

Params: lat lon coords eircode generate_map tile_source
POST /api/garden-pins/batch

Get all delivery pins within a radius. Returns front and back pins for every building. Precompute first for instant results.

Params: lat lon radius_m garden_type min_score generate_map
POST /api/classify

Classify a GPS coordinate as front garden, back garden, building, road, etc. Returns classification, surface type, and delivery suitability score (0-100).

Params: lat lon tile_source
POST /api/precompute

Precompute classification for a large area. Single-pass processing — 500m radius in ~1-3 minutes. Subsequent batch queries are near-instant.

Params: lat lon radius_m skip_cache
GET /api/cache/stats

Cache statistics: fast cache entries, precomputed areas, total pins stored.

DELETE /api/cache

Clear all caches — fast point cache and precomputed area data.

POST /api/shutdown

Gracefully shut down the server. Sets shutdown flag so in-flight tile fetches abort, then sends SIGTERM.
